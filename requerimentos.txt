#API REST #

Contexto da API : TRANSPORTE DE CARGA E PESSOAL, ROTAS DE ENTREGA PARA UM GRUPO DE CIDADES LOCAIS
Unidade e objetivo: ROTAS UNIDIMENSIONAIS, DESEJA-SE CONHECER AS MELHORES ROTAS EM ALGUMAS CIDADES 

Input/Entrada: ENTRADA GRAFO DIRECIONADO ONDE UM NÓ REPRESENTA UMA CIDADE E UMA LINHA A ROTA, PESO DA LINHA  = DISTÂNCIA 
A MESMA ROTA NÃO APARECE DUAS VEZES, CIDADES ORIGEM E DESITNO SÃO DIFERENTES 
Tipo da entrada: A ROTA É DADA EM JSON 

API DEVE SALVAR O GRAFO, RECUPERAR O GRAFO, 
ENCONTRAR TODAS AS ROTAS DISPONÍVEIS DADA UMA ORIGEM E UMA DESTINO (DADO NÚMERO MAX DE PARADAS)

HTTP METHOD:  POST
